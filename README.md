# SKP NYPRO Dukla — wersja wielojęzyczna

## Pliki językowe
- `index.html` — polski
- `en.html` — angielski
- `sk.html` — słowacki
- `ua.html` — ukraiński

## Dodane funkcje
- przełącznik języka PL / EN / SK / UA
- znaczniki SEO `hreflang`
- osobne tytuły, opisy i frazy SEO dla każdej wersji
- sekcja o przypomnieniu terminu kolejnego badania po wykonanym badaniu
- Twoje logo, baner i zdjęcia budynku

## Ważne po publikacji
W plikach HTML, `robots.txt` i `sitemap.xml` zamień:
`https://twoj-login.github.io/nypro/`
na faktyczny adres strony.

## GitHub Pages
Wgraj całą zawartość paczki do repozytorium GitHub i włącz Settings → Pages.
